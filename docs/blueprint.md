# **App Name**: BillEase Pro

## Core Features:

- Invoice Entry: Allow users to input product rates, quantities, and customer names.
- Tax Calculation: Enable manual entry of CGST and SGST percentages for calculation.
- Total Price Calculation: Automatically calculate the total price including taxes.
- Invoice Formatting: Format the invoice in a modern, user-friendly layout.
- Invoice Header: Add "MATESHWARI EXPORTS\nMfrs. & Wholesale : All types of Jeans & Cotton Pa" at the top of each invoice.
- Invoice Download: Allow users to download the invoice as a PDF file.
- Invoice Saving: Enable users to save generated invoices within the application.

## Style Guidelines:

- Primary color: Light blue (#ADD8E6) for a professional and clean look.
- Background color: Off-white (#F5F5F5) to ensure readability and reduce eye strain.
- Accent color: Green (#32CD32) to highlight important actions and totals.
- Body and headline font: 'Inter' for a modern and readable appearance.
- Use simple, professional icons for actions like download and save.
- Design a clear, structured layout with distinct sections for input, calculations, and invoice display.
- Use subtle animations to indicate successful actions, like saving or downloading an invoice.
